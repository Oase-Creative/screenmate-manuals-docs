## Overview

This playbook describes the complete process for SEO optimization at Oase Creative. SEO is outsourced to Daniel Al Seffar.

## Step 1: Form Check

Check if the SEO form is completely filled out and all required assets are present (keywords, target audience info, existing content).

## Step 2: Intake & Goals

Discuss goals with the client via call or Slack: rankings, visibility, traffic, conversions.

## Step 3: Handover to Daniel

Provide all assets to Daniel via Slack channel.
Daniel performs: keyword research, technical analysis, content plan, and optimizations.

## Step 4: Implementation

Implementation happens on the website / Shopify / documentation environment.

## Step 5: Revision

Any feedback from client is implemented.

## Workflow Summary

Form → Intake → Handover → Analysis → Implementation → Feedback → Completion.
