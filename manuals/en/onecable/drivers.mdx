## Driver Installation (Windows)

If you are using the USB-A method, or if the image does not appear immediately, you may need to install a driver.

<Steps>
  <Step title="Step 1: Download Driver">
    Use the included USB stick or scan the QR code to download the driver.
    
    <div className="w-48">
      <img src="/images/Screenmate%20-%20OneCable%20-%20Handleiding%20images/oneCable%20qr%20code%20driver%20install.png" alt="QR Code Driver" className="rounded-lg border border-gray-200" />
    </div>
  </Step>
  <Step title="Step 2: Installation">
    Open the installation file and follow the on-screen instructions.
    <img src="/images/Screenmate%20-%20OneCable%20-%20Handleiding%20images/Driver%20installatie%20voor%20Windows.png" alt="Driver Installation Windows" className="rounded-lg border border-gray-200" />
  </Step>
  <Step title="Step 3: Restart">
    Restart your computer after installation to apply the changes.
  </Step>
</Steps>
