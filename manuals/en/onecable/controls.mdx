## Controls & OSD

<Note>
This section covers the physical buttons on the monitor and how to navigate the settings menu.
</Note>

### Physical Buttons

Typically, the monitor has buttons for:
- Power On/Off
- Menu / Select
- Volume / Brightness adjustment
- Return / Exit

### OSD (On-Screen Display) Menu

Press the Menu button to access settings for:
- Brightness & Contrast
- Color Temperature
- Audio Settings
- Input Selection
- Language
