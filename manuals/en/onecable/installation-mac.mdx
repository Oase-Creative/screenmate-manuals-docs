## Driver installation for macOS

### Step 1: Connect USB stick

Insert the USB stick into your MacBook. Open the driver folder that appears on the desktop.

<img src="/images/Screenmate%20-%20OneCable%20-%20Handleiding%20images/installation-images/screenmate-onecable-installation-mac-step-1.png" alt="Step 1: Connect USB stick" className="rounded-lg border border-gray-200 dark:border-gray-700" />

### Step 2: Open MAC-os folder

Click on the MAC-os folder.

<img src="/images/Screenmate%20-%20OneCable%20-%20Handleiding%20images/installation-images/screenmate-onecable-installation-mac-step-2.png" alt="Step 2: MAC-os folder" className="rounded-lg border border-gray-200 dark:border-gray-700" />

### Step 3: Open RacerUSB folder

Click on the RacerUSB folder.

<img src="/images/Screenmate%20-%20OneCable%20-%20Handleiding%20images/installation-images/screenmate-onecable-installation-mac-step-3.png" alt="Step 3: RacerUSB folder" className="rounded-lg border border-gray-200 dark:border-gray-700" />

### Step 4: Install driver

Follow the steps to install the driver.

<img src="/images/Screenmate%20-%20OneCable%20-%20Handleiding%20images/installation-images/screenmate-onecable-installation-mac-step-4.png" alt="Step 4: Install driver" className="rounded-lg border border-gray-200 dark:border-gray-700" />

### Step 5: Open System Settings

Go to System Settings > Privacy & Security > Screen Recording.

<img src="/images/Screenmate%20-%20OneCable%20-%20Handleiding%20images/installation-images/screenmate-onecable-installation-mac-step-5.png" alt="Step 5: System Settings" className="rounded-lg border border-gray-200 dark:border-gray-700" />

### Step 6: Check UsbDisplay

Ensure UsbDisplay is turned on.

<img src="/images/Screenmate%20-%20OneCable%20-%20Handleiding%20images/installation-images/screenmate-onecable-installation-mac-step-6.png" alt="Step 6: Check UsbDisplay" className="rounded-lg border border-gray-200 dark:border-gray-700" />

### Step 7: Add UsbDisplay (if needed)

If UsbDisplay is not listed, click the + sign to add it.

<img src="/images/Screenmate%20-%20OneCable%20-%20Handleiding%20images/installation-images/screenmate-onecable-installation-mac-step-7.png" alt="Step 7: Add UsbDisplay" className="rounded-lg border border-gray-200 dark:border-gray-700" />

### Step 8: Open RacerUSB

Go to Applications, select RacerUSB, choose UsbDisplay, and click Open.

<img src="/images/Screenmate%20-%20OneCable%20-%20Handleiding%20images/installation-images/screenmate-onecable-installation-mac-step-8.png" alt="Step 8: Open RacerUSB" className="rounded-lg border border-gray-200 dark:border-gray-700" />

### Step 9: Configure displays

Go to Displays and configure the screens as desired.

<img src="/images/Screenmate%20-%20OneCable%20-%20Handleiding%20images/installation-images/screenmate-onecable-installation-mac-step-9.png" alt="Step 9: Configure displays" className="rounded-lg border border-gray-200 dark:border-gray-700" />

<Note>
Scan the QR Code in the front of the booklet to install the driver if you cannot open the driver file.
</Note>
